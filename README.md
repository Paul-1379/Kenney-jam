# Sokoscale
A sokoban game made for the 2026 kenney game jam in godot 4.

The game also match two bonus challenges of the jam: 'no text' and 'mouse-only' 
---

---

You solve puzzle by pushing small and big boxes, but the big boxes can only by pushed if you are tall. 

The game is mouse only, click to move the character and click on the character to change your size.
